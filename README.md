# OWASP Juice Shop Bug Report
This repository contains bug reports created while testing the OWASP Juice Shop application.

## Objective
- Practice identifying reproducible software defects.
- Write professional bug reports with clear reproduction steps.
- Document expected vs actual behaviour.
- Capture supporting screenshots.

## Application
OWASP Juice Shop

## Number of Bugs Reported
5

## Report Format
Each bug report contains:
- Bug ID
- Module
- Environment
- Description
- Preconditions
- Steps to Reproduce
- Expected Result
- Actual Result
- Severity
- Priority
